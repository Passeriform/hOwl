# Physical Constructs

## Keyword
Any space separated phrase

## Hierarchy
Activator: `<Enter-Tab>`

The hierarchy root becomes heading/root-node.

## Sibling
Activator: `<Enter>`

## Points
Activator: `<Enter>`

Special kind of sibling with no leaf properties.

## Comparison
Compare siblings for same stuff. Append rest to individual points.

## Definition
Activator: ` - `

## Implication
Activator: `=>`

## Preview Definition
Activator: `()`

## Flow
Activator: `->`

## Variables
Activator: `create`

Assume every variable exists.
Create scoped variables.

## Loop
Activator: `such that`

Child hierarchy runs for loop.

## Iterator
Activator: `iterate`

## Lists
Activator: `[]`

## Checkboxes
Activator: `[X]` `[ ]`

## Question
Activator: `?`
